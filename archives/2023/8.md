有时候需要在终端应用中直接打开 URL，可使用以下快捷键。

| 操作 | 快捷键 |
| --- | --- |
| 选择 URL | 按住 Shift-Command 键并连按 URL |
| 打开 URL | 按住 Command 键并连按 URL |
| 选择完整文件路径 | 按住 Shift-Command 键并连按路径 |
| 选择整行文本 | 点按该行三下 |

更多 [Mac 上“终端”中的键盘快捷键](https://support.apple.com/zh-cn/guide/terminal/trmlshtcts/mac)。
