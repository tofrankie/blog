## 一、什么是 Stylish ？

先看效果图，Stylish 把百度首页变成了这样子:

![](https://cdn.jsdelivr.net/gh/tofrankie/blog@main/images/2026/1/1767260288529.png)

Stylish 是一款浏览器插件，目前适用于 Chrome、Safari、Firefox 等主流浏览器。原理是通过替换网页本身的 CSS 样式来达到网页美化的效果。

[Stylish](https://userstyles.org/styles/browse) 官网上有非常非常多的样式供用户下载，不仅能让百度、新浪微博变得极简、讨人喜欢，还能给新浪网、腾讯网、Google 、YouTube 、Facebook 、Wikipedia 等主流网站「换肤」。

如果你懂 CSS 知识，还可以自定义样式，做一个属于自己的网页效果，也可以在已有样式的基础上进行修改完善。

## 二、安装使用 Stylish 插件

下面我将以 Chrome 浏览器为例子，其他浏览器的使用方法大同小异，不太浏览器效果也可能略有不同。

1. 下载安装插件：前往 [Chrome 应用商店](https://chrome.google.com/webstore/category/extensions?hl=zh-CN)下载安装。亦可去 [插件网](http://www.cnplugins.com/devtool/stylish/)（不用翻墙）下载。

2. 在浏览器扩展程序页面启用 Stylish 插件后，前往 [Stylish 官网](https://userstyles.org/styles/browse)下载模板。

3. 可以对已安装模板进行修改、禁用、删除、更新等。

![](https://cdn.jsdelivr.net/gh/tofrankie/blog@main/images/2026/1/1767260488349.png)

4. 快捷启用、禁用、修改样式。

![](https://cdn.jsdelivr.net/gh/tofrankie/blog@main/images/2026/1/1767260529777.png)
