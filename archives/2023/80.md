今天创建了一个新项目

```json
{
  "scripts": {
    "start": "webpack-dev-server",
    "build": "webpack --progress --colors"
  },
  "devDependencies": {
    "webpack": "^5.42.0",
    "webpack-cli": "^4.7.2",
    "webpack-dev-server": "^3.11.2"
  }
}
```

然后搜查了一番，有部分人是删除了 `node_modules`，重新 install 可以跑起来，但这是不对的。

解决方法：

> Install the lastest webpack-cli and use `webpack serve` to run webpack dev server instead `webpack-dev-server`. [webpack/webpack-dev-server #3304](https://github.com/webpack/webpack-dev-server/issues/3304)

安装最新的 `webpack-cli`，使用 `webpack serve` 来代替 `webpack-dev-server`。

```json
{
  "scripts": {
    "start": "webpack serve",
    "build": "webpack --progress"
  },
  "devDependencies": {
    "webpack": "^5.42.0",
    "webpack-cli": "^4.7.2",
    "webpack-dev-server": "^3.11.2"
  }
}
```

改完就能愉快地玩耍了...
