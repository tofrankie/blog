在使用 npm 安装包时，遇到 npm Warn name@x.x.x No repository field.

意思是你的 `package.json` 缺少 `repository` 仓库字段。可采用以下两种方式解决 WARNING。

有两种解法：

1. 添加仓库地址：

```json
{
  "repository": {
    "type": "git",
    "url": "http://github.com/xxx/xxx.git"
  }
}
```

2. 设为私有（如果该包是不发布的情况下）

```json
{
  "private": true
}
```
