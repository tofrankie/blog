## 背景

此前在 Windows 设置中进行系统更新，总是安装错误 - 0xc1900101，导致升级不成功。

## 方法

1. 前往 https://www.microsoft.com/en-us/software-download/ 下载 Windows 镜像（按需下载）
2. 打开 Parallels Desktop 的控制中心，点击对应镜像的设置按钮，打开配置窗口
3. 切换至硬件选项卡，找到 CD/DVD，在「源」处选择你的 ISO 镜像文件
4. 启动 Windows 虚拟机，它会自动挂载一个虚拟光驱（例如 D:）
