在 [skills.sh](https://skills.sh) 收集了 7000+ 的 Skill，支持一众 AI Editor：

- Claude Code
- Cursor
- Windsurf
- Antigravity
- Kiro
- Trae
- Cline
- CodeX
- ...

导入非常简单：

```bash
$ npx skills add <owner/repo>
```

> [add-skill](https://github.com/vercel-labs/add-skill)

比如：`npx add-skill vercel-labs/agent-skills`
